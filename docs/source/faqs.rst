1. ValueError: to_mesh(): Mesh 'xxx' is in editmode
Get error from:

>>> pt111.replace('Pt', 'Au', range(20))

Solution: Exit the ``Edit`` mode.


2. RuntimeError: Operator bpy.ops.object.select_all.poll() failed, context is incorrect
Get error from:

>>> co.translate(t)

Solution: Exit the ``Edit`` mode.
