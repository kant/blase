
===================
Hydrogen bond
===================

To build up hydrogen bond, the value for ``search`` should be set to ``5``. To change setting for a bond pair by:

>>> h2o.bondsetting['O-H'].min = 2.2
>>> h2o.bondsetting['O-H'].max = 3.2
>>> h2o.bondsetting['O-H'].search = 5
